Banner
EDC festival

Así fue el proceso de creación del banner para el festival de música electrónica EDC, en el cual se desarrolló una propuesta visual pensada para transmitir la energía, el color y la atmósfera vibrante que caracteriza a este tipo de eventos. Durante el proceso se exploraron diferentes ideas, estilos y composiciones hasta lograr un diseño que combinara elementos llamativos, una paleta de colores intensa y una estructura visual equilibrada. El objetivo fue crear un banner que no solo fuera estéticamente atractivo, sino que también representara la identidad del festival y captara la atención del público.

El Reto

Crear este banner presentó varias dificultades durante el proceso de diseño. Uno de los principales retos fue lograr una buena integración entre todos los elementos visuales, como el búho, las flores, el logotipo y el fondo de la ciudad, para que se vieran parte de una misma composición. También fue complejo equilibrar los colores intensos y contrastantes sin que el diseño se viera saturado. Además, fue necesario cuidar la jerarquía visual para que el logo y la información del evento fueran fáciles de leer, mientras los elementos decorativos mantenían la estética llamativa del festival.

La solución

La solución fue trabajar en la armonización de los elementos visuales mediante ajustes de color, iluminación y composición. Se aplicó una paleta cromática coherente para que el búho, las flores, el logotipo y el fondo se integraran de forma equilibrada. También se reorganizaron los elementos para mejorar la jerarquía visual, asegurando que el logotipo y la información del evento fueran claros y legibles. Finalmente, se realizaron ajustes de contraste y tamaño para mantener un diseño llamativo sin perder claridad ni equilibrio visual.

Mi trabajo

Mi trabajo fue pensar y crear desde cero el banner para el festival, desarrollando el concepto visual, seleccionando los elementos gráficos y construyendo una composición que representara la energía y la identidad del evento. Para lograrlo, fue necesario explorar diferentes ideas, colores y formas hasta encontrar una estética que transmitiera el ambiente vibrante del festival. Además, me encargué de integrar todos los elementos del diseño, como el logotipo, las ilustraciones y el fondo, cuidando que mantuvieran un equilibrio visual y que el resultado final fuera atractivo, claro y coherente.
